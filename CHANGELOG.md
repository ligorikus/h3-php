# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-20

### Added
- Initial release
- Pure PHP implementation of H3 geospatial indexing system
- Support for H3 resolution 0-15
- Cell, Vertex, DirectedEdge types
- Value Objects: LatLng, CellBoundary, GeoPolygon, GeoLoop, CoordIJ
